Favourite Technologies

Most of my proyects we develop by using at least one of this programming languages as well as some kind of Integrated Development Environment.
This list shows those I prefer working with, not all nor less but my favourites and with whom I like to work:

    Programming Languages

  • DataBases SQL: Nowadays data science is on the rise because of the new technologies like AI are going to change many aspects of our lifes, even in our works as developers. The first step in order to reach to understand this new tecniques are dealing with Big Data, so that massive data does not mean an inconvinient to you.

  • Java: High level of skills designing programs implemented on Java. Most of my personal proyects were develop on this programming language because of its popularity and how powerfull it is. Although is getting old, it makes a necesity to any kind of programmer to know about how to implement programs and read code on Java.

  • HTML, CSS & JavaScript: Although I am not a huge fan of web design, even when I created this website on my own, I admit that it is also important to at least understand how a website is developed and to acknowledge the work involved. It was fun doing it alone and the effort pays off as having your own website is a huge bonus.

    • IDEs

  • Eclipse: It is the main tool that I used to do my Java projects, so my level of knowledge of this tool is quite high. Easy to use and very flexible, it is perfect for small tasks such as homework and practice deliveries.

  • IntelliJ IDEA: On the other hand, many subjects were tought using this IDE because of its elaborate proyect design and usefull correction marks. Personally I would not say it is my favourite but it resoult to be quite an advantage in some ocassions.

  • Visual Studio Code: The facility to change of programming language makes VSC one of a kind. Little more to say about it.

  • Oracle tools: A great sum of subjects required using tools provided by Oracle such as "SQL Developer Data Modeler" or even tools made on Java. It is vital for a computer science engineer to have some skills on this kind of tools.
